## Описание проекта

Графическое WPF-приложение «Калькулятор чаевых». Принимает сумму счёта, количество гостей и процент чаевых (0%, 5%, 10%, 15%), рассчитывает:
- сумму чаевых;
- итоговую сумму (счёт + чаевые);
- сумму на каждого гостя (если гостей > 1).
