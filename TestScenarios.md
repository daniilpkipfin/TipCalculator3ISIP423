## ТС-001. Расчёт с 10% чаевых, 1 гость

| Поле | Значение |
|------|----------|
| **Цель** | Проверить корректный расчёт чаевых при одном госте |
| **Предусловия** | Приложение запущено |
| **Шаги** | 1. Ввести сумму счёта: **1000** |
| | 2. Ввести количество гостей: **1** |
| | 3. Выбрать **10%** |
| | 4. Нажать «Рассчитать» |
| **Ожидаемый результат** | Чаевые: 100,00 руб. Итого: 1100,00 руб. Блок «на гостя» не отображается |
| **Фактический результат** | Отображается: Сумма счёта: 1000,00 руб., Чаевые: 100,00 руб., Итого: 1100,00 руб. Блок «На каждого гостя» скрыт. |
| **Статус** | Пройден ✅ |
